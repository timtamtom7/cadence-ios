# App Store Listing - Cadence

## App Name
**Cadence** — Focus Timer

## Subtitle
Focus. Breathe. Achieve.

## Description
Cadence is a beautiful focus timer that helps you build lasting habits through focused work sessions, partner accountability, and streak tracking.

**Features:**
- Beautiful breathing orb for guided focus
- Partner mode for accountability
- Streak tracking to build habits
- Weekly challenges to stay motivated
- Team focus sessions
- Achievement badges
- AI-powered focus insights
- Multiple ambient soundscapes
- Dark mode optimized
- iPad optimized layout

**Focus Modes:**
- Deep Focus (45 min)
- Pomodoro (25 min)
- Meditation (15 min)
- Custom durations

**Soundscapes:**
- Rain
- Forest
- Ocean
- Cafe
- Fireplace
- White noise
- And more...
